# Hardware build guide

The full hardware build guide — bill of materials, safety warnings, step-by-step disassembly, battery upgrade, and speaker installation — lives in the top-level [README](https://github.com/GreerBK/raycanary#hardware-build-guide).

It is intentionally kept in the README rather than split across mdBook pages so that:

- It's the first thing visible when someone lands on the repo
- The safety warnings can't be missed by clicking past a chapter intro
- The "verify yourself" markers stay in-line with the steps that depend on them

If you've reached this page from the mdBook, follow that link.
